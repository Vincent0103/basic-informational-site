Deep Work

Prelude

In these modern days, the ability to get into a flow state and do deep work is becoming more rare and rare.

So the new thing in this new economy to be able to succeed and show real value, it is by leveraging the power of deep work and applying it to our lives.

The Concept of Deep Work

Deeply working on a specific task for long periods of time without interruption, the more time i'm into deep work the more the focus is becoming intense (there is this rule that says that we first need to work for atleast 20 mins with no interruption on a task to fully be in a flow state).
